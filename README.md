# GpuInstanceGrass
## 介绍

unity GpuInstance合批的可摆动的草,支持设置草颜色

已支持：

- GpuInstance合批
- 可以对每颗草设置颜色 位置 缩放
- 草分块（运行时只渲染附近格子的草）
- 编辑模式批量刷草
